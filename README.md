## Contact Form Setup

The site uses [Formester](https://formester.com) for contact form management:

- **Form Dashboard:** [https://app.formester.com/forms/76536/email_notifications](https://app.formester.com/forms/76536/email_notifications)
- **Features:**
  - Auto-height responsive form
  - Custom transparent design matching site theme
  - Direct email notifications
  - Spam protection
  - Mobile-optimized

### Form Colors Used:
- Primary: `#2563EB` (button color)
- Background: Transparent (`#FFFFFF00`)
- Input fields: `#FFFFFF1A` (10% white)
- Placeholder text: `#FFFFFF99` (60% white)
- Input text: `#FFFFFF` (white)
- Border: `#FFFFFF33` (20% white)

### To Update Form:
1. Log into Formester dashboard (link above)
2. Modify form fields, styling, or notifications
3. Changes reflect automatically on the site
4. Form submissions go to configured email
